## 安装
在本地环境中运行此项目之前，请确保已安装以下依赖项：
1. Python 3.11 或更高版本
2. 其他依赖项（如有，请参考 `requirements.txt`）

安装步骤如下：
```bash
# 克隆项目
git clone https://github.com/your-repo-url/toy.git

# 进入项目目录
cd toy

# 安装依赖
pip install -r requirements.txt

```

### 安装 playwright

```
playwright install
```
